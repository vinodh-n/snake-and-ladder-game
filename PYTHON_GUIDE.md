# Python Snake and Ladder Game - Quick Start Guide

## Installation

### Prerequisites
- Python 3.6 or higher
- Terminal/Command Prompt

## Running the Game

### Option 1: Direct Execution (Easiest)
```bash
python snake_and_ladder.py
```

### Option 2: With Python 3 Explicitly
```bash
python3 snake_and_ladder.py
```

## Running Tests

### Option 1: Using unittest
```bash
python -m unittest test_snake_and_ladder.py -v
```

### Option 2: Direct execution
```bash
python test_snake_and_ladder.py
```

## Game Instructions

### Starting the Game
1. Run the script
2. Enter names for Player 1 and Player 2 (or press Enter for defaults)
3. Follow the prompts
